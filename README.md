# Katyusha

A web app launcher for demo and test

## Sign in modes

1. Manually sign-in
2. Microsoft AAD Single-factor sign-in
3. Basic auth

## Additional features

1. Share password
2. Optional pin code to unlock for added security
3. New version download notification
4. Post launch tasks (advanced RPA sequence)
